§B — 10 procedural styles (different aesthetics where SVG actually shines)

These don't try to replicate hand-illustrated emoji. They lean into SVG's geometric strengths: sacred symmetry, low-poly facets, stained glass borders, pixel grids, line work, gradient glows. Each style is its own design language. Default colors shown; all recolor via fill swap.

Production model: Each picked moon = one SVG vendored at static/vendor/moons/{source}/{name}.svg. Owner picks per-moon variant + tint color. For multi-moon worlds: a "randomize" button picks a different variant for each new moon. License attribution lives in one CREDITS.md: each emoji set's license line + a "procedural designs CC0" line.
