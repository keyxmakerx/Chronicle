Render fixed. Last version showed solid squares — my filter bug, not yours. The game-icons SVGs are a white icon on an opaque black square, and my old filter flattened the whole thing. Now using mix-blend-mode: screen to drop the black box, on a dark sky-gradient backing (the real environment these live on). You should now see actual icons.

Candidate base icons from game-icons.net (CC-BY 3.0, one line in CREDITS.md covers all). Each shown three ways: silhouette (luminous white, the default sky-band look), warm (gold-tinted day mood), crimson (eclipse / blood-moon mood). Tints are CSS filters — the real build recolors per state the same way, and adds animated rays / glow / sparkle particles on top.
