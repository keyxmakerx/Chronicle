# Bestiary Plugin

## Purpose

The Community Bestiary enables instance-wide sharing of homebrew creature statblocks.
Users publish creatures from their campaigns, and all authenticated users on the
instance can browse, search, rate, favorite, and import them into their own campaigns.
Instance admins can moderate flagged content.

## Key Architectural Decisions

- **Instance-scoped, not campaign-scoped.** Routes live at `/bestiary/*`, not under
  `/campaigns/:id/`. Publications are visible to all authenticated users.
- **Statblock JSON stored as-is.** The `statblock_json` column stores the complete
  creature definition. Level, organization, and role are denormalized into indexed
  columns for efficient filtering.
- **Slug-based URLs.** Publications are accessed by slug (`/bestiary/ashen-wyrm`)
  for SEO and shareability.
- **Atomic rating aggregation.** Rating sum/count are stored on the publication row
  and updated atomically to avoid expensive aggregate queries on read.
- **Per-user rate limiting.** Token bucket middleware keyed by user ID, applied at
  the route level with limits per the design doc.

## File Layout

```
internal/plugins/bestiary/
  embed.go          - Embedded migration files
  model.go          - Domain types, DTOs, cross-plugin interfaces
  repository.go     - MariaDB persistence (BestiaryRepository interface)
  service.go        - Business logic (BestiaryService interface)
  handler.go        - HTTP handlers (thin, delegates to service)
  routes.go         - Route registration (user + admin)
  ratelimit.go      - Per-user rate limiting middleware
  migrations/       - SQL migration files
  .ai.md            - This file
```

|--------|------|---------|------------|

| GET | /bestiary/trending | Trending | 200/min |
| GET | /bestiary/newest | Newest | 200/min |
| GET | /bestiary/top-rated | TopRated | 200/min |
| GET | /bestiary/most-imported | MostImported | 200/min |
| GET | /bestiary/favorites | ListFavorites | - |
| GET | /bestiary/creators/:userId | CreatorProfile | - |
| GET | /bestiary/:slug | Show | - |
| GET | /bestiary/:slug/statblock | GetStatblock | - |
| GET | /bestiary/:slug/reviews | ListReviews | - |

## Cross-Plugin Dependencies

| Interface | Wraps | Purpose |
|-----------|-------|---------|
| `UserFetcher` | `auth.AuthService.GetUser` | Creator profile display names |
| `EntityCreator` | `entities.EntityService.Create` | Import creates campaign entities |
| `CampaignRoleChecker` | `campaigns.CampaignService.GetMember` | Import verifies Scribe+ role |

All adapters are defined in `internal/app/routes.go`.
