# Mentions Widget

## Purpose

Provides inline @mention functionality for the TipTap rich text editor.
When users type `@` in the editor, a search popup appears showing matching
entities from the campaign. Selecting an entity inserts a styled mention link
that preserves entity metadata through JSON round-trips.

## Tier

**Widget** -- pure frontend extension for the editor widget. No Go backend.

## Files

| File | Role |
|------|------|
| `internal/widgets/mentions/templates/` | Templ mount-point (currently empty) |
| `static/js/widgets/editor_mention.js` | TipTap mention extension + popup UI |

## Dependencies

- **Uses:** Entity search API (`GET /campaigns/:id/entities/search?q=...`)
- **Uses:** TipTap editor instance (provided by editor widget)
- **Used by:** Editor widget (`editor.js`) via `Chronicle.MentionExtension`

## Frontend Widget

### Integration

Not mounted via `data-widget`. Instead, the editor widget loads it as a TipTap
extension when `data-campaign-id` is set:

```js
// In editor.js
if (config.campaignId) {
    extensions.push(Chronicle.MentionExtension(campaignId));
}
```

### Key Classes

| Export | Description |
|--------|-------------|
| `Chronicle.MentionExtension` | Factory returning TipTap extension config |
| `Chronicle.MentionLink` | Extended Link mark preserving mention `data-*` attributes |
| `MentionPopup` (internal) | Dropdown UI with keyboard nav and debounced search |

### Behavior

1. User types `@` → popup appears near cursor.
2. Debounced search (200ms, min 2 chars) queries entity search API.
3. Results show entity name, type name, and type color indicator.
4. Arrow keys navigate, Enter selects, Escape dismisses.
5. Selected entity replaces `@query` with styled anchor: `<a class="mention-link" data-mention-id="..." href="...">@EntityName</a>`.
