# Title Widget

## Purpose

Displays the entity name as a prominent page heading with conditional action
buttons (edit, clone, delete) based on user permissions. Rendered as a Templ
component — no JavaScript.

## Tier

**Widget** -- pure server-rendered Templ component. No client-side JS.

## Files

| File | Role |
|------|------|
| `internal/widgets/title/templates/` | Templ mount-point component |
| Entity `show.templ` `blockTitle()` | Inline rendering in entity profile page |

## Dependencies

- **Uses:** Campaign context (permissions, roles)
- **Uses:** Entity data (name, ID)
- **Used by:** Entity profile page layout system (entities plugin)

## Rendering

Part of the entity profile layout block system. The block type `"title"` maps
to the `blockTitle()` Templ function in the entity show template.

### Output

```html
<h1 class="text-3xl font-bold">{EntityName}</h1>
<!-- Conditional action buttons for Scribe+ users -->
<a href="/campaigns/{cid}/entities/{eid}/edit">Edit</a>
<button hx-post="/campaigns/{cid}/entities/{eid}/clone">Clone</button>
<button hx-delete="..." >Delete</button>  <!-- Owner+ only -->
```

### Visibility Rules

| Action | Minimum Role |
|--------|-------------|
| View name | Any member |
| Edit link | Scribe |
| Clone button | Scribe |
| Delete button | Owner |
